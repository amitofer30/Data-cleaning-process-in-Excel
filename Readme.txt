This folder contains an example of data cleaning. The example is not completely comprehensive, but it contains the basic principles of data cleaning. The PDF file contains a description and explanation of all the operations performed on the data.

The dataset under consideration offers a diverse range of information about
Netflix’s offerings, including movies and TV shows. It comprises several key fields such as ’show id’,
’type’, ’title’, ’director’, ’country’, ’date added’, ’release year’, ’rating’, ’duration’, and ’listed in’. These
fields encompass categorical, numerical, and textual data, presenting a rich opportunity for comprehensive data cleaning and preprocessing.
